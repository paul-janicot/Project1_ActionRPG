# an Action/Adventure game

School project at [ArtFx](5https://artfx.school/le-campus/montpellier/) 😎  
First project for the 2nd year working with 3rd year Game designers/programers


### Games features
- Assyrian aesthetics 
- protagonist : an apothicary with a sick sister 🥳
- Environmeent : city (babylone) and hanging garden, include a lot vegetation
- player fantasy : sneak and attack with a grappling hook

### Inspirations
- Tunic, A short Hike, chants of Scennar
